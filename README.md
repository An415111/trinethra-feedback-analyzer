# Trinethra Supervisor Feedback Analyzer

## Project Overview
AI-assisted supervisor transcript analysis tool built for DeepThought internship assignment.

The application analyzes supervisor feedback transcripts using Ollama LLM and generates structured draft analysis for psychology interns.

## Features
- Supervisor transcript input
- AI-powered transcript analysis
- Structured rubric scoring
- Evidence extraction
- KPI mapping
- Gap analysis
- Follow-up question generation
- Human review warning system

## Tech Stack
Frontend:
- React
- Vite
- Axios

Backend:
- Node.js
- Express
- Axios

LLM:
- Ollama
- llama3.2

## Why Ollama?
I used Ollama because the assignment required local LLM execution without cloud APIs. Ollama provides easy local API integration and fast experimentation for prompt engineering.
